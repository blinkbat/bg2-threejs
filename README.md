# BG2-ThreeJS

Baldur's Gate 2-inspired isometric RPG built with React + Three.js + TypeScript + Vite.


## Setup

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` — Start dev server with HMR
- `npm run build` — Production build
- `npm run lint` — ESLint
- `npm run test` — Run tests
